## Climate Resilience and Rural Livelihoods: A Difference-in-Differences (DiD) Analysis

Conducted an econometric impact evaluation to measure the effectiveness of climate resilience policies on agricultural and rural economic outcomes. Applied Difference-in-Differences (DiD) regression models on panel data to estimate causal policy effects while controlling for confounding factors. Performed data cleaning, exploratory data analysis, statistical testing, and model validation using Python and econometric techniques. Analyzed indicators including agricultural GDP, rural employment, and productivity metrics to quantify the economic implications of climate adaptation initiatives. The project demonstrates the application of causal inference methods, policy evaluation frameworks, and data-driven decision-making in environmental and development economics.
